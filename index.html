Ramen Group

EJS Bootstrap Node.js Express MongoDB Mongoose Javascript Puppeteer RESTful routing NodeMailer JWT
See Demo

About this project

Ramen Group Web Application is a full-stack RESTful Node and Express app made for helping users to find their favorite ramen stores around them. Features include the ability to view all ramen stores scraped from google map, create an account, edit your profile, add a new favorite store you find around you, leave comments, and rate stores. Also, user can find nearest ramen stores in their city or find the best rating ramen store around them.
