About this project

Ramen club web application is a full-stack React and Node.js app made for helping users to find their favorite ramen stores around them. Features include the ability to view all ramen stores crawled from google map, create an account, edit your profile, add a new favorite store you find around you, leave comments, and rate stores. Also, the website allow users to find nearest ramen stores in their city implementing geohash and Uber h3 geo-spacial indexing.

About this project

HowHow Parser is a backend node app which generate spoken sentence using parsed YouTube video from famous YouTuber HowHow. The video contains all the chinese character pronunciation and I extract all the time-pronunciation relationship and write a parser accordingly. The app features typing a sentence in the form and the app cuts the video and mergers the sentence pronounced by the youtuber for you.
